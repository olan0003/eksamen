GENDE WEB

grundlægende

I dette tema arbejdede vi med grundprincipperne af kodning, primært af html og css kodning. vi startede med at lære om grid, og de forskellige måder man kunne bruge grid på. Derefter begyndte vi at eksmpementere med css design, hvor vi prøvede nogle simple ting af. vi arbejdede også med forskellen på desktop og phone, og det var her vi begyndte med Mediaqueries, og øvede os i at ændre på grid set ups, alt efter skærm bredden. Alt dette førte frem til vores første opgave, som var at sætte en hjemmeside op, udfra nogle wireframes valgt af lærren. Vi fik selv lov til at lege med css designet på hjemmesiden, da det vigtigste i dette tema var selve kodningen af htmlén og grid-layouts
