Grundlægende Web

I dette tema arbejdede vi med grundprincipperne af kodning, primært html og css kodning. Vi startede med at lære om grid, og de forskellige måder man kunne bruge grid på. Derefter begyndte vi at eksperimentere med css design, hvor vi prøvede nogle simple ting af. Vi arbejdede også med forskellen på desktop og phone, og det var her vi begyndte med Media queries, og øvede os i at ændre på grid set ups, alt efter skærm bredden. Alt dette førte frem til vores første opgave, som var at sætte en hjemmeside op, ud fra nogle wireframes valgt af vores lærer. Vi fik selv lov til at lege med css designet på hjemmesiden, da det vigtigste i dette tema var selve kodningen af htmlén og grid-layouts
